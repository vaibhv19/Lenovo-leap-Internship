# SkillPath

SkillPath is a responsive React landing-page application built with Vite and React Router.
It demonstrates modern web development concepts such as reusable UI components, client-side routing, controlled form handling, and responsive layout.

## Project Structure

- `index.html` - Application entry point.
- `src/main.jsx` - React application bootstrap.
- `src/App.jsx` - Main app shell with router configuration.
- `src/components/` - Reusable UI components like `Navbar`, `Footer`, `HeroSection`, `FeatureCard`, `TestimonialSection`, and `CTABanner`.
- `src/pages/` - Page views for the landing page, about page, contact page, and 404 page.

## Features

- Responsive landing page experience
- Multi-page navigation with `react-router-dom`
- Reusable React components for a clean UI structure
- Contact form with validation and success feedback
- Modern Vite-based development workflow

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn available

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Open the local dev server URL shown in the terminal to preview the app.

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

## Technologies

- React 18
- React Router DOM 6
- Vite 5

## Notes

This project is designed as a polished demo website for learning and presenting modern frontend development patterns.
It is a great starting point for enhancements such as animation, a real contact backend, or theme customization.
