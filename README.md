## 🚀 Getting Started & Local Setup

Each directory inside this repository runs independently. Follow these steps to set up and run any of the assignments locally:

### 📋 Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed (version 18+ is recommended).

### ⚙️ Installation & Running

1. Clone this repository to your local system and enter the repository folder:
   ```bash
   git clone <repository-url>
   cd "Lenovo leap internship"
   ```

2. Select the assignment directory you want to run (e.g., `Assignment 1-draft 2`):
   ```bash
   cd "Assignment 1-draft 2"
   ```

3. Install all necessary dependencies:
   ```bash
   npm install
   ```

4. Run the local development server:
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) in your browser to view the application.

5. Compile a production build:
   ```bash
   npm run build
   ```

6. Run the linter to verify code syntax and style consistency:
   ```bash
   npm run lint
   ```
