## 🚀 Quick Start — Run any project locally

Prerequisites: Node.js 18+ and npm.

1. Open a terminal and change into the project folder you want to run. Examples:

```powershell
cd "d:/vaibhav gupta/Coding/Projects----Internship/Lenovo leap internship/Assignment 1-draft 2"
```
or

```powershell
cd "d:/vaibhav gupta/Coding/Projects----Internship/Lenovo leap internship/MILESTONE 7- Assignment"
```

2. Install dependencies:

```powershell
npm install
```

3. Start the dev server:

```powershell
npm run dev
```

4. Open the address shown by Vite (typically `http://localhost:5173`) in your browser.

5. To create a production build:

```powershell
npm run build
```

6. To run linters (if the project includes lint scripts):

```powershell
npm run lint
```
